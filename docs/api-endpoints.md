# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Pins

- `GET /api/pins`
- `POST /api/pins`
- `GET /api/pins/:id`
- `PATCH /api/pins/:id`
- `DELETE /api/pins/:id`

### Boards

- `GET /api/boards`
- `POST /api/boards`
- `GET /api/boards/:id`
- `DELETE /api/boards/:id`

### Follows

- `GET /api/followers`
- `GET /api/following`
- `POST /api/following`
- `DELETE /api/following/:id`
